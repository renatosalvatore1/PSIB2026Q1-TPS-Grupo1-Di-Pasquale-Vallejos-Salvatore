# Como ejecutar nuestro proyecto:
Se debe ejecutar el codigo desde "main.py". Primero se descargarán los registros y luego se realizará el preprocesamiento y extraccion de caracteristicas de cada uno de los registros. Los resultados obtenidos se exportan en la carpeta "resultados_para_analisis" en formato xlsx. Los resultados estadisticos se encuentran en la carpeta "Resultados Graphpad". Para utilizar la interfaz, esta debe ejecutarse desde "Interfaz.py". Desde la misma se puede elegir un registro y visualizarla en ventanas de 2,5,10,20 y 30 segundos, visualizarle pre y poste preprocesamiento y visualizar el PSD completo de cada canal y sus caracteristicas
